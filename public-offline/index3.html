🎯 4元數 Node 紀錄表 - 學生操作手冊

歡迎來到 4元數 Node 紀錄表！這是一個幫助你進行矩陣與節點數值推演的視覺化工具。所有的紀錄都會即時同步到雲端，讓你與組員們可以輕鬆共編與討論。

本手冊會教你如何建立分頁、新增數值、以及利用疊加參照層來推導規律。

🚀 1. 介面基本操作

建立與切換「Lv 分頁」

在最上方的控制列，你可以看到各種按鈕：

  • 「+ 新增 Lv 分頁」：當你完成當前層級 (Level) 的推演後，按此按鈕會建立新的一頁（例如從 Lv 0 推進到 Lv 1）。
  • 分頁切換器：利用「◀ 上一頁」與「下一頁 ▶」按鈕，在不同的推演步驟間來回檢視。

在分頁中增加資料列

  • 「⬆️ 往上新增一列 Node (+2)」：點擊後，會在現有的樹狀圖頂端再往上長出一整列矩陣，每一列的節點數量會比下一列多 2 個，方便你展開推演脈絡。
  • 「⬇️ 刪除最上列 (還原)」：如果按錯了或者推演方向有誤，點擊即可移除最上面的一列。

🧮 2. 編輯與標記矩陣 (Nodes)

畫面上的每一個方塊，都代表一個 2x2 的矩陣，內部包含四個數字 (a, c 在第一排，b, d 在第二排)。

輸入數值

  1. 直接點擊 Node 內的白色小框框即可打字輸入代數或數字。
  2. 鍵盤快速導覽：使用鍵盤的 「上、下、左、右方向鍵」，可以在格子之間快速跳躍移動，不需要一直用滑鼠點擊。

標記節點狀態 O 與 X

當你需要標記哪個節點符合特定規律條件（勝/敗點），你可以：

  1. 將滑鼠停留在該方塊 (Node) 的上方。
  2. 方塊上方會浮現 綠色「O」 與 紅色「X」 小按鈕。
  3. 點擊對應按鈕，方塊就會變色，方便你視覺化統整規律！
    註：如果標記錯誤，再點擊一次同一個按鈕就會恢復白色。

🔍 3. 疊加參照層 (Reference Layer) - 進階功能

為了幫助你觀察圖形的生長規律與遞迴關係，系統會在 Lv 1 以上 的分頁中啟用「參照層」機制！

唯獨參照列

在樹狀圖的最頂端，你會看到一列半透明、無法編輯的數字框，上方標示著 [ 參照自 Lv ... ]。
這代表這一列的數字，是系統自動幫你從先前的推演步驟中提取出來的，讓你可以「對照上一層的最後一列」來繼續往上推演，不需要一直切換前後頁確認數字。

半透明「疊加層」開關

在「參照自 Lv...」那行文字的正下方，有一個 「顯示 Lv {當前層-1}」 的核取方塊 (Checkbox)。

  • 打勾開啟：原本乾淨的背景上，會浮現許多半透明縮小的舊節點，這些是系統幫你把「上一頁的矩陣」依照精準的幾何位置直接疊加在現在的空隙中。
  • 你可以藉由這個疊加殘影，更清楚地看見「上一層的圖形如何發展、位移成了這一層」。
  • 當畫面太雜亂時，隨時取消打勾就能瞬間隱藏它們。

💾 4. 存檔、匯入與輸出圖片

  • 儲存雲端 (自動化)：你在格子內輸入的任何東西，只要停下動作 0.6 秒，就會自動變成「黃燈 儲存中...」，然後轉為「綠燈 已連線同步」存進共編資料庫，不須手動按檔儲存。
  • 「💾 備份 (JSON)」：下載一份 .json 檔案到你個人的電腦中備份，以防萬一。
  • 「📂 匯入覆蓋」：上傳你之前備份好的 .json 檔案，這會直接把這個進度強行覆蓋當前的畫面，並立刻同步給其他組員（操作前請與組員確認喔！）。
  • 「🖼️ 匯出當前畫面」：系統會幫你把整個樹狀圖變成一張超高畫質的去背圖片 (.png) 下載下來，寫報告、投影片時非常好用！
